Picture-less observations are useful when you've recorded a species you saw but were not able to take a photograph of it.

Numbering specimens (observations) is for scientific use, such as for submitting plant specimens to a herbarium.

The increment method starts from a number you specify and adds one for each specimen. Eg. starting with '300': 300, 301, 302, etc.

The append method adds an incremented number to the text you specify. Eg. using 'Dave2013': Dave2013-1, Dave2013-2, etc.
